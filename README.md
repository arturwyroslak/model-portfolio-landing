# Model Portfolio Landing

Zaawansowany, responsywny szablon landing page dla portfolio modelki. Zawiera sekcje: hero, galerie, about, services, booking i kontakt. Gotowy do podłączenia do CMS lub statycznego hostingu.

Instrukcje:
- Umieść swoje zdjęcia w katalogu assets/
- Dostosuj treści w index.html
- Możesz dodać backend do obsługi formularza booking
